# 📝 پروژه Todo با Next.js 15 و Zustand

این یک پروژه‌ی ساده‌ی Todo است که با استفاده از **Next.js 15** ساخته شده و مدیریت وضعیت آن توسط **Zustand** انجام می‌شود.
هدف این پروژه یادگیری و پیاده‌سازی یک اپلیکیشن Todo سبک و سریع با ساختار مدرن است.

---

## 🚀 ویژگی‌ها

* ✅ افزودن Todo جدید
* 🗑️ حذف Todo
* ✏️ ویرایش Todo
* 💾 ذخیره‌سازی داده‌ها در **localStorage** (حتی بعد از رفرش باقی می‌ماند)
* ⚡ مدیریت State سبک و بهینه با Zustand
* 🧭 ساختار پروژه ساده و قابل گسترش

---

## 🧰 تکنولوژی‌های استفاده‌شده

* [Next.js 15](https://nextjs.org/)
* [Zustand](https://docs.pmnd.rs/zustand/getting-started/introduction)
* JavaScript / TypeScript
* LocalStorage

---

## 📦 نصب و راه‌اندازی

ابتدا پکیج‌ها را نصب کنید:

```bash
npm install
```

سپس پروژه را در حالت توسعه اجرا کنید:

```bash
npm run dev
```

و از طریق آدرس زیر وارد اپلیکیشن شوید:

```
http://localhost:3000
```

---

## 🧭 ساختار کلی پروژه

```
📂 project-root
 ├─ 📂 app              # صفحات اصلی پروژه
 ├─ 📂 components       # کامپوننت‌های UI مثل TodoStatus, AddTodoForm
 ├─ 📂 lib              # لایبرری های استفاده شده
     ├─ todo-store.ts   # استور Zustand
 ├─ 📂 types            # تایپ‌ها (ITodo)
 ├─ package.json
 ├─ README.md
```

---

## 🧠 نکته فنی

* داده‌ها پس از هر تغییر به صورت خودکار در LocalStorage ذخیره می‌شوند.
* هنگام رفرش صفحه داده‌ها باقی می‌مانند.
* کامپوننت‌ها فقط در سمت کلاینت رندر می‌شوند تا از بروز خطای Hydration جلوگیری شود.
